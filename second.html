Question-1: What are the different ways to select an element in DOM?

There are 5 ways to select element in DOM getElementById(“id name”)
getElementsByClassName(“class name”)
getElementByTagName(“specific tag”)
querySelectorBy(“tagName / # id /. className”)
querySelectorByAll(“tagName / # id /. className”)

Question-2: What is the difference among innerHTML, innerText and textContent?

InnerHTML provides all text content of parent and child node along with tag (for exapmle like a chocolate box without wrapping paper)
textContent provides all text of parent and child node including hide element and space (for example all chocolate including space)
innerText gives raw text of parent and child node without space and hide text

Question-3: What is event delegation in the DOM?

Event delegation means declare event listener upon common ancestor instead node that provide advantages such as saving memory, simple code, dynamic action etc. (for example ‘smoking causes cancer’ the dialogue no need to attach on individual cigeratte instead labelling on package)

Question-4: What is event bubbling in the DOM?

Event bubbling means propagation of event signal from target to root of DOM object.(for example  government budget set for specific work but budget being shared like bubbling )
